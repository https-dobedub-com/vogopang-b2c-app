# vogopang-b2c-app

Vogopang B2C mobile application built with Expo and React Native.

## Tech Stack

- Expo
- React Native
- TypeScript

## Run

```bash
npm install
npm run start
```

You can also run platform shortcuts:

```bash
npm run android
npm run ios
npm run web
```

## Notes

- App config: `app.json`
- Entry file: `App.tsx`
- Native folders (`ios`, `android`) are ignored and can be generated later if needed.
